# 監造附錄六合一產出系統部署說明

## 建議路線

### 免費快速產生網址：Streamlit Community Cloud

這個專案已經準備好 `app.py`、`requirements.txt`、`packages.txt`，可以放到 GitHub 後用 Streamlit Community Cloud 部署。

注意：Streamlit Community Cloud 是 Linux 環境，不能執行本機 Microsoft Word COM。程式仍可產出 Word，但頁碼/欄位更新會少掉本機 Word 那一層校正。如果要跟你電腦上的輸出完全一致，建議用 Windows 主機部署。

步驟：

1. 建立一個 GitHub repository。
2. 上傳這個資料夾內容，但不要上傳 `.venv`。
3. 到 Streamlit Community Cloud 建立 app。
4. Repository 選你的 GitHub repo。
5. Main file path 填 `app.py`。
6. Deploy 後把網址給同事。

### 最穩、最接近本機輸出：Windows 主機

如果一定要維持 Word 版面、頁碼、欄位更新最精準，請用一台 Windows 電腦或 Windows Server 跑這個 Streamlit app，並裝好：

- Python
- Microsoft Word
- 專案依賴套件

啟動方式：

```powershell
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

再用公司內網、VPN、Cloudflare Tunnel、ngrok 或正式網域把 `8501` 連出去。
